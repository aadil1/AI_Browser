SafeBrowse Privacy Policy

Last Updated: December 27, 2025

We believe in transparency and your right to privacy. This policy details how SafeBrowse handles your data.

1. Data Collection

SafeBrowse does not collect, store, or sell your personal data.

  • We check webpage content only when you explicitly interact with the extension.
  • Content is processed solely for safety analysis (detecting prompt injections).
  • We do not store your browsing history.

2. Data Processing

When you use the extension, data is transmitted to an API endpoint that you configure.

  • If you use the default provided endpoint, data is processed in ephemeral memory and discarded immediately after analysis.
  • We do not use your data to train AI models.

3. Local Storage

The extension stores the following data locally on your device using Chrome's Storage API:

  • Your API Configuration (Endpoint URL and API Key).
  • Your user preferences (e.g., Dark Mode settings).

4. Third-Party Sharing

We do not share any data with third-party advertisers, trackers, or data brokers.

5. Contact Us

If you have questions about this policy or the extension's security, please contact the developer via the GitHub repository or support channel.
