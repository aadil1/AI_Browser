## Reporting a Vulnerability

We take the security of `SafeBrowse` properly. If you believe you have found a security vulnerability, please report it to us as described below.

**Do not report security vulnerabilities through public GitHub issues.**

### Procedure

Please send an email to **hello@safebrowse.io**.

You should receive a response within 24 hours. If for some reason you do not, please follow up via email to ensure we received your original message.

### Prefer Sealed Reports

For highly sensitive vulnerabilities, please encrypt your report using PGP if possible, or request a secure drop box link in your initial email.
